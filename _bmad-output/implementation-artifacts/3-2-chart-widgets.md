# Story 3.2: Chart & Visualization Widgets

## Description
Build a library of chart widgets including line charts, bar charts, pie charts, area charts, and metric cards with Recharts.

## Acceptance Criteria
- Line chart widget with multi-series support
- Bar chart with grouping and stacking
- Pie/donut chart with drill-down
- Area chart with gradient fills
- Metric card with sparkline and trend indicator
- All charts support dark/light mode
